# Spring OpenTelemetry Example

This example demonstrates integrating Pyroscope profiling with OpenTelemetry in a Spring Boot application using the
`otel-pyroscope-spring-boot-starter` for auto-configuration.

It is adopted
from [otel-profiling-java/examples/with-otel-library](https://github.com/grafana/otel-profiling-java/blob/d02d563978bb55ca8ea485c38bb6a918f35d7e76/examples/with-otel-library).
